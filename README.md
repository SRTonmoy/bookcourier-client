# BookCourier

A full-stack book ordering and management system with role-based dashboards.

## Roles
- User: Browse books, place orders
- Librarian: Add and manage books
- Admin: Manage users and books

## Tech Stack
- Frontend: React, Tailwind, DaisyUI
- Backend: Node.js, Express
- Database: MongoDB
- Auth: Firebase Authentication

## Features
- Role-based protected routes
- Secure JWT authentication
- Responsive UI
- REST API integration
